# Bus Ticket Booking System

A DBMS-based Bus Ticket Booking System developed using Java and MySQL, enabling users to search buses, book tickets, manage passenger details, and handle reservations efficiently.

## Features
- User Registration and Login
- Bus Search and Availability Check
- Ticket Booking
- Passenger Details Management
- Reservation Tracking
- Database Management using MySQL

## Technologies Used
- Java
- MySQL
- JDBC

## Project Objective
To provide an efficient and user-friendly system for managing bus ticket reservations and passenger records.
